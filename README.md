# TODO App - Backend

## Description

This is the API for the TODOAPP. The frontend interface was created using React, you can [take a look here](https://github.com/leobritob/youtube-todoapp-frontend).

## Preview
![TODOAPP SWAGGER API](./preview-swagger.png)

## Before Installation

Create a .env file and fill with your database credentials.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
